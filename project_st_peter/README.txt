Project St Peter Readme File

To run the project run the following command into R console:
runUrl("https://github.com/Gabrieloana91/my_script/archive/master.tar.gz",subdir = "project_st_peter", host = "172.x.x.x", port = 9001)

The host needs to be set up using the following steps:

run(cmd)
ipconfig
insert as "host" the "IPv4 Address" (172.x.x.x)
port can be set as whichever port you want.
